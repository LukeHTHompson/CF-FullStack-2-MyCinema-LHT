Welcome to MyCinema

This application allows you to pull information about countless movies by accessing an API created by Luke Thompson.

API Endpoints

List All Movies

Description: This endpoint will GET a list of all movies in the database. This list will be displayed to the requester.
URL: "/movies"
Method Type: GET
Parameters: N/A
Request Format: GET http://localhost:8080/movies
Response Format: JSON Array of Movies (or maybe HTML movie list?)
Example Request: GET http://localhost:8080/movies
Example Response:
[
{ title: Wall-E,
director: Andrew Stanton},
{ ... }
]

Read Movie Data

Description: GET detailed information on a single movie.
URL: "/movies/:title"
Method Type: GET
Parameters: Title - this is the movie to return details on.
Request Format: GET http://localhost:8080/movies/[title]
Response Format: List of details about the movie requested.
Example Request: GET http://localhost:8080/movies/Catch%20Me%20If%20You%20Can
Example Response:
__Title: Catch Me If You Can __Description: Follow Detective Carl Hanratty (played by Tom Hanks) as he pursues one of the most notorious counterfeiters in US history - Frank Abignale Jr.
__Genre: Crime Fiction
__Director:
__Image URL:
__Featured: Yes

Update Movie Favorite

Description: Adds a movie to a user's list of favorite movies.
URL: "/:userId/favorites/:title/add"
Method Type: POST
Parameters: userId - this is the currently logged in user's ID.
title - this is the movie to be added to favorites.
Request Format: POST http://localhost:8080/[userId]/favorites/[title]/add
Response Format: [title] has been added to your favorites!
Example Request: POST http://localhost:8080/12231996/favorites/Up/add
Example Response: "Up" has been added to your favorites!

Update Movie Unfavorite

Description: Removes a movie from a user's list of favorite movies.
URL: "/:userId/favorites/:title/remove"
Method Type: DELETE
Parameters: userId - this is the currently logged in user's ID.
title - this is the movie to be removed from favorites.
Request Format: DELETE http://localhost:8080/[userId]/favorites/[title]/remove
Response Format: [title] has been removed from your favorites.
Example Request: DELETE http://localhost:8080/12231996/favorites/Up/remove
Example Response: "Up" has been removed from your favorites.

Read Genre Data

Description: Tells the user more about a particular film genre.
URL: "/genres/:genre"
Method Type: GET
Parameters: Genre - this is the genre that the user wants more info on.
Request Format: GET http://localhost:8080/genres/[genre]
Response Format: Serve static HTML with details on the genre requested.
Example Request: GET http://localhost:8080/genres/horror
Example Response: Redirects user to webpage with info on Horror genre.

Create User

Description: Creates a new user account registered with the site.
URL: "/createUser"
Method Type: POST
Parameters:
Request Format: POST http://localhost:8080/createUser
The body of the request must contain a JSON Object cotaining at least a value for the key "username".
Response Format: Account successfully created for [user].
Example Request: POST http://localhost:8080/createUser
Example Response: Account successfully created for ElThompson.

Update Username

Description: Updates the username for the currently logged in user.
URL: "/:userId/:name"
Method Type: POST
Parameters: userId - this is the ID of the currently logged in user.
Request Format: POST http://localhost:8080/[userId]/[name]
Response Format: Your username has successfully been updated to [name].
Example Request: POST http://localhost:8080/12231996/Luke%20Thompson
Example Response: Your username has successfully been updated to Luke Thompson.

Delete User

Description: Deletes the account of the currently logged in user.
URL: "/:userId/delete"
Method Type: DELETE
Parameters: userId - this is the ID of the currently logged in user.
Request Format: DELETE http://localhost:8080/[userId]/delete
Response Format: Your account was successfully deleted.
Example Request: DELETE http://localhost:8080/12231996/delete
Example Response: Your account was successfully deleted.
